# Suporte a navegadores da interface de pessoa usuária do OpenBao

O OpenBao atualmente suporta todos os navegadores "evergreen" e atualizados.
Os seguintes navegadores são suportados:

- Chrome
- Firefox
- Safari
- Microsoft Edge

:::danger

**Aviso**: O uso de um navegador não suportado, como o Internet Explorer 11 (IE
11), pode causar degradação na funcionalidade de alguns recursos e, em alguns
casos, os recursos do OpenBao podem não funcionar.
Recomendamos o uso de um dos navegadores suportados listados para a interface de
pessoa usuária do OpenBao.

:::

Observe que o OpenBao, em consonância com a posição da Microsoft sobre o IE 11,
não oferece mais suporte ao Internet Explorer 11 (IE 11).
Para obter mais informações sobre o IE 11, consulte o
[site de suporte](https://docs.microsoft.com/en-US/lifecycle/faq/internet-explorer-microsoft-edge)
da Microsoft.
