# Funcionamento interno do OpenBao

Esta seção aborda o funcionamento interno do OpenBao e explica os detalhes
técnicos de como o OpenBao funciona, sua arquitetura e propriedades de
segurança.

**Nota:** O conhecimento do funcionamento interno do OpenBao é recomendado, mas
não obrigatório para usar o OpenBao.
No entanto, se você estiver operando o OpenBao, recomendamos que compreenda seu
funcionamento interno.
