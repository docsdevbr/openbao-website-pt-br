# OpenBao Auto-Auth methods

Auto-auth is a mechanism used by OpenBao Agent and OpenBao Proxy to authenticate
to OpenBao in an automatic manner, given a set of parameters allowing the
authentication. Please see the sidebar for available methods and their
usage/configuration.
